❓ שאלות נפוצות

מה זה קובץ JSON ולמה צריך אותו?

קובץ JSON הוא ארגז שבו ארוזה כל העבודה שלכם — כל הסימונים, הקטגוריות, החבילות והערות המחברת. הוא מאפשר למרצה לבדוק את העבודה בכלי אוטומטי. אין צורך לפתוח אותו — רק להעלות ל-Moodle.

שכחתי לשמור ויצאתי — אבדו הנתונים?

לא! האפליקציה שומרת אוטומטית בדפדפן. פתחו שוב את אותו דפדפן באותו מחשב.

עברתי מחשב — מה עושים?

לפני שמחליפים מחשב, הורידו גיבוי JSON. במחשב החדש — פתחו את האפליקציה ולחצו ייבוא JSON כדי לטעון את העבודה.

הקובץ JSON נראה כמו "קוד" — תקין?

כן! זה בדיוק איך שקובץ JSON נראה. סגרו והעלו אותו כמו שהוא.

לא מוצא/ת את כפתור הגיבוי — איפה?

הכפתור נמצא רק בשלב 5 (קריאה) — הכפתור האחרון בסרגל. ודאו שהגעתם לשלב האחרון.
